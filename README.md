# Quiz Application in C

## Description
This is a simple Quiz Application made using the C programming language.

## Features
- User enters their name
- Multiple-choice questions
- Score calculation
- Displays final result

## Concepts Used
- Variables
- Input/Output (printf, scanf)
- if-else
- Basic C programming

## How to Run
1. Compile the code.
2. Run the program.
3. Answer the quiz questions.
4. View your final score.
